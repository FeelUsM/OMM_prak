план такой:
скрипт на матлабе
создает текстовый(ые) файл(ы) со входными данными
и функцией system через командную строку винды запускает скомпилированный перед этим exeшник
этот экзешник читает входной файл(ы) 
решает задачу 
и опять создает файл с выходными данными
скрипт на матлабе читает этот файл
и рисует график

екзешник компилится из исходника(ков) на с++
в данном случае кмпилятором MSVC
хотя можно и любым другим (анпример mingw-ом)
(
для простой прогонки перекомпилировать ничего не надо,
а вот для заданных задач - придется править исходники и перекомпилировать
)

потребуется: матлаб и MSVC
их можно скачать с рутреккера (давно качал, ссылку не помню, но через гугл наверняка легко ищется)
или через мой яндекс-диск:
матлаб: https://yadi.sk/d/xD5T29cuTKLxJ
MSVC: https://yadi.sk/d/GtobZ3ldTKM5a

пошаговое руководство решения задачи X такое
*все установить
*открыть проэкт (.sln) в папке X
*открыть исходник и отредактировать в нужных местах
*нажать F7 (кмпиляция без запуска, т.к. файлов входных данных еще нету)
*запустить матлаб
*открыть в корне файл X.m
*отредактировать параметры входных данных
*нажать F5 (запуск) (и в первый раз нажать ОК на заданный вопрос)
*оно нарисует графики, и может даже сохранит их
